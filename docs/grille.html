Grids

3 systèmes de grilles sont disponibles :

  • .tb-grid Basée sur un table layout.
  • .fl-grid Basée sur un float left layout.
  • .il-grid Basée sur un inline-grid layout.

Gouttière

La taille de la gouttière est par défaut équivalente au rythme vertical.
Cette valeur est gérée par la var(--gutter)

Colonnes

Les grilles permettent diverses compositions avec un système de 2 à 7 colonnes imbricables, avec gouttières par défault.
Les taille des colonnes sont gérés grâce aux class span--*.
La taille de la colonne s'ajoute en option. Ex: span--1-3.
Astuce : cette class est optionnelle (voir exemple 1).
Une class span--auto est également disponible.

Blocs options

  • .no-gutter pour une grille sans colonnes
  • .half-gutter pour des tailles de gouttières réduites de moitier
  • .va-top .va-middle .va-top pour modifier l'alignement vertical des colonnes.
    Fonctionne uniquement avec les table-grid et il-gird.
